# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.1.x | ✅ |

## Reporting a Vulnerability

Please report security issues using GitHub Security Advisories for this repository
(Security tab → "Report a vulnerability"). If you cannot use GitHub, open a public
issue without sensitive details and request a private reporting channel.

We aim to acknowledge reports within 3 business days and provide a remediation
plan or timeline within 14 days.
